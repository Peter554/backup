Gratis-Paketaufkleber per DHL

Drucken Sie einen fertig ausgefüllten und bereits bezahlten DHL-Paketaufkleber aus, den Sie per E-Mail erhielten, kleben Sie Ihn auf das Paket und geben Ihr Paket an einer DHL-Annahmestelle ab (max. Paketgewicht 31,5 kg). Den Versandaufkleber können Sie außerdem erneut herunterladen, wenn Sie Ihren Verkauf im Nutzerprofil unter "meine Verkäufe" aufrufen. Wenn Sie keinen Drucker besitzen, zeigen Sie den auf dem Paketaufkleber vorhandenen QR-Code über ihr Smartphone in der Postfiliale. Dort wird Ihnen ein Paketaufkleber gedruckt. 

Regulärer Versand

Falls Sie keine der kostenlosen Versand-Möglichkeiten nutzen können (z.B. bei Paketen aus dem Ausland), adressieren Sie bitte Ihr Paket per Hand. Diese Versandart ist nicht kostenlos.
